## Overview
This involves the development of a digital inventory management system for **Petro Special**, a company specializing in the trade of automotive oils, tires, and batteries.
This project aims to automate the tracking of stock levels and sales.

## Features (v1.0)
- Add items to inventory (Oils, Tires, Batteries).
- View current stock status.
- C++ Console Application.
